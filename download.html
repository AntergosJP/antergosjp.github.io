ダウンロード

Antergos はフリー (無料・自由) な OS です。自由にダウンロードとコピーの作成ができます。 ISO イメージをダウンロードして、 CD や DVD 、または USB メモリに書き込むだけで簡単に楽しめます。

Antergos の LiveUSB を作成するために推奨する方法を Wiki にまとめています: HOW TO: Create a working Live USB

テスト版イメージ

私たちはテスト用にイメージの自動生成を行っています。これらは毎週日曜日にリリースされています。これには最新のパッケージと GitHub の安定版ブランチ由来の最新の Cnchi が含まれます。

私たちはテスト用イメージを1週間のみサーバーに保存しています。新しいイメージを生成する際に、古いものは削除されます。ビルドログはイメージと同じディレクトリから見ることができます。必要に応じてご確認ください。
